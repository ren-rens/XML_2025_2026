## Упътване
Една CDATA секция представя текстово съдържание на XML елемент и позволява включване на специални XML знаци.

Синтаксис: ```<![CDATA[ ..... ]]>```

Пример: ```<![CDATA[ if(x > y && x > max) { max = x;} ]]>```
